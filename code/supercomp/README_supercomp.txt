README_supercomp

You should only launch submit_master_array_vr.sh. This can either
	1. Launch precompiled models
	2. Launch a compilation of models.

To do 1, sample code is:
bash submit_master_array_vr.sh 1 35 airt log_lambda normal /work/compagna/vr-1519665825

To do 2, sample code is: 
bash submit_master_array_vr.sh 1 1 /work/compagna/vr-4280462060
